# Language Reference

Core mappings:

- `highkey` / `lowkey`: const / mutable declarations
- `is`: assignment
- `bet` / `cap`: true / false
- `sus` / `nah`: if / else
- `cook` / `secure`: function / return
- `sendit` / `fumbled drama`: try / catch
- `yoink ... outta ...`: imports
- `binge ... in`: for-in loop
- `spam`: while loop
- `yap`: print/log output

Indentation rules:

- Block scope is indentation-based.
- Tabs and spaces are both allowed.
- One tab is normalized as two spaces.
- Mixed indentation style at the same depth is a syntax error.
