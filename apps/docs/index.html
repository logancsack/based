  lowkey token is vault.get("API_TOKEN")

Troubleshooting

  • Mixed indentation style at the same depth: use one style per block depth.
  • Cannot reassign highkey: mutate values declared with lowkey.
  • No module provider configured for yoink: run via CLI/server runtime.
