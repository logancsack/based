Quickstart

pnpm install
pnpm build
pnpm --filter based-lang run based -- check examples/hello.fr
pnpm --filter based-lang run based -- run examples/hello.fr
pnpm --filter based-lang run based -- serve --entry examples/webhook.fr --port 3000

Scripts are indentation-based and use the .fr extension.

Stdlib Reference

network

yoink network outta stdlib
lowkey res is network.get("https://api.example.com")
yap res.status
yap res.body

vault

yoink vault outta stdlib
lowkey token is vault.get("API_TOKEN")

discord/slack

yoink discord, slack outta stdlib
discord.send("Deployment shipped")
slack.send("Webhook ping complete")

Set env vars: DISCORD_WEBHOOK_URL, SLACK_WEBHOOK_URL.

Self-Host Deployment Guide (Node + Docker)

# Local server
based serve --entry examples/webhook.fr --port 3000 --secret supersecret

# Docker
docker build -t based-local-runner .
docker run -p 3000:3000 -e DISCORD_WEBHOOK_URL=... based-local-runner

Webhook requests should include x-based-signature when a secret is configured.

Cookbook

  • Webhook handler: parse payload and return transformed JSON.
  • Discord notifier: send a message when an event is received.
  • Slack notifier: publish status updates to a channel webhook.
  • API bridge: fetch from an API with network.get and forward the result.
yoink network, discord outta stdlib

cook main(payload)
  lowkey weather is network.get("https://api.example.com/weather")
  discord.send("Weather update: " + weather.status)
  secure { ok: bet, status: weather.status }

Troubleshooting and Error Catalog

  • Mixed indentation style at the same depth: do not mix tabs and spaces in one block depth.
  • Cannot reassign highkey: use lowkey for mutable vars.
  • No module provider configured for yoink: run through CLI/server runtime, not raw parser only.
  • network call blocked: configure allowlist if host restrictions are enabled.
