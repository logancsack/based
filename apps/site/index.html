Based
Docs Playground GitHub

Meme-first language. Real utility.

Build webhooks and bots in pure slang.

Based is a dynamic scripting language with indentation syntax, batteries-included stdlib, and a local-first toolchain for shipping fast.

Read Docs Open Playground 
npm install -g based-lang

Language Core

`sus`, `nah`, `cook`, and `secure` map to familiar control flow while keeping code readable and fun.

CLI + Self-Host

Use `based run`, `based check`, `based fmt`, and `based serve` to execute scripts and host webhook handlers.

Agent-Ready Docs

Ship both human HTML and markdown mirrors at `/docs-md`, plus `/llms.txt` and `/llms-full.txt` for AI indexing.
